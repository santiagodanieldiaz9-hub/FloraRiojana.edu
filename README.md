# Flora Autóctona de La Rioja

Presentación interactiva creada con React, Vite y Tailwind CSS.

## Comandos

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Estructura

- `src/App.jsx`: componente principal de la presentación.
- `src/main.jsx`: punto de entrada.
- `src/index.css`: estilos globales con Tailwind.
